Developer guide

Getting started

解析服务提供了四种接口，其中三个基于http的，分别为/http /ftp /url, 他们主要根据数据来源划分。另外一种是基于thrift的远程调用方式。

Http Method

  • Host

    10.46.126.36:8765

  • method

    POST

  • post params:

    (1) tid    [string] [required]      eg:525ff8934fde442268000004

(2) html   [string] [required]      eg:<div>harvest</div>

(3) render [string] [not required]  eg:true

(4) params [string] [not required]  eg:{"basepos": 10}

    NOTE: 高级参数：params 是用来向模板中注入变量的，例如用basepos来确定此页结果是从几个开始 ；params的值是json string，务必确保传入的值能够被JSON.parse（）处理，否则可能导致解析失败

Url Method

  • Host

    10.46.126.36:8765

  • method

    POST|GET

  • post params:

    (1) tid    [string]  [required]      eg:525ff8934fde442268000004

(2) url    [string]  [required]      eg:http%3A//www.baidu.com

(3) render [string]  [not required]  eg:true

    NOTE: 如果用GET方式请求，请注意编码特殊字符（eg： url_encode or escape）
